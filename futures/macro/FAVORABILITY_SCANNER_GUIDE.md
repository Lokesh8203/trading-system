## 📊 Global → MCX Conversion Guide

### Gold
- Global: GC=F (COMEX Gold, $/oz)
- MCX: GOLDM (Gold Mini, ₹/10g)
- Conversion: 1 oz = 31.1g = 3.11 × 10g
- Example: $4,880/oz ≈ ₹15,700/10g (adjust for $/₹ rate ≈ 83)

### Silver
- Global: SI=F (COMEX Silver, $/oz)
- MCX: SILVER (₹/kg)
- Conversion: 1 oz = 31.1g, 1 kg = 1000g
- Example: $82/oz ≈ ₹2,637/kg

### Crude Oil
- Global: CL=F (WTI Crude, $/barrel)
- MCX: CRUDEOIL (₹/barrel)
- Conversion: Direct (both in barrels)
- Example: $83/barrel ≈ ₹6,889/barrel ($/₹ = 83)

**Note:** MCX prices include import duties, storage, so slight premium over global.

## 🚀 How to Use This Scanner

### Daily/Weekend Routine

**1. Run Scanner (Anytime)**
```bash
python3 futures/macro/favorability_scanner.py
```

**2. Check Rankings**
- Score 75+: High conviction (take up to 40% allocation)
- Score 60-74: Moderate (take 20-30%)
- Score <60: Wait

**3. Check Returns Projection**
- Best/Avg/Worst case
- Probability profitable
- Is expected value positive?

**4. Check Gap Risk**
- <50: Low (commodities)
- 50-70: Medium
- >70: High (indices) → reduce allocation

**5. Make Decision**
- High score + positive EV + acceptable gap risk = TAKE
- Otherwise = WAIT

### Weekend Planning

**Friday Evening:**
1. Run scanner
2. Note high-conviction opportunities (75+)
3. Check if they have high gap risk (>70)
4. Decide: Enter now or wait for Monday?

**For indices (gap risk 80):**
- Enter 1/3 position only
- Check GIFT Nifty Monday 8:30 AM
- Add 1/3 more if gap favorable
- Exit if gap >2% against

**For commodities (gap risk 40):**
- Can enter full position
- 24hr markets, can monitor/exit
