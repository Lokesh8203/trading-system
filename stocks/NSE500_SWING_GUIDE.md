## 🎯 What Scanner Looks For

### **1. Narrow Range 7 (NR7)** ⭐⭐⭐
```
Your #1 proven pattern

What it is:
- Last bar has narrowest range of last 7 bars
- Range < 2.5% (tight consolidation)
- Coiled spring ready to expand

Why it works:
- Volatility contraction → Volatility expansion
- Clear stop loss (below consolidation)
- High win rate when in Stage 2

Entry: At close or slight above (1%)
Stop: 2% below consolidation low
Target: 5-10% move
```

### **2. Narrow Range 4 (NR4)** ⭐⭐
```
Tighter version of NR7

What it is:
- Last bar has narrowest range of last 4 bars
- Even tighter coil

Same logic as NR7, just shorter lookback
```

### **3. 200 EMA Ride** ⭐⭐⭐
```
Your #2 proven pattern

What it is:
- Stock in Stage 2 uptrend
- Pulls back to 200 EMA
- Holds above EMA (within 2%)
- Bullish reversal candle

Why it works:
- 200 EMA acts as support in uptrend
- Institutional buying at key level
- Defined risk (stop below EMA)

Entry: Just above 200 EMA (1-2%)
Stop: 3% below 200 EMA
Target: 8-15% move
```

## 📊 Trade Management

### **Entry:**
```
1. Set GTT order at entry zone high
2. Valid for 1 week
3. If not triggered, reassess setup
4. Never chase (max 2% slippage)
```

### **Stop Loss:**
```
1. Place immediately after entry
2. Use GTT stop loss order
3. Never move stop closer (only trail up)
4. Exit without emotion if hit
```

### **Targets:**
```
Target 1 (Conservative):
- Exit 50% position
- Move stop to breakeven
- Let rest run

Target 2 (Aggressive):
- Exit remaining 50%
- Or trail stop aggressively

If stock extends beyond T2:
- Trail stop with 50 EMA
- Let winners run
```

### **Time Stop:**
```
If no movement after 2 weeks:
- Reassess setup
- If Stage breaks, exit
- If consolidating, hold
- Max hold: 4 weeks (unless big winner)
```

## 🚨 Red Flags (Skip These)

### **Market Conditions:**
```
❌ Market in Stage 4 (bear market)
❌ Nifty breaking key supports
❌ High VIX (> 20) - too volatile
❌ Global events (war, crisis)
```

### **Stock Conditions:**
```
❌ Earnings in next 7 days (skip)
❌ Pending results/news
❌ Sudden gap up > 5% (too late)
❌ Low liquidity (< ₹50L/day)
❌ Promoter selling
❌ Regulatory issues
```

### **Setup Conditions:**
```
❌ Range > 3% (not tight enough for NR)
❌ Price > 30% from 52W high (too extended)
❌ 200 EMA declining (not Stage 2)
❌ R:R < 1.5:1 (poor risk/reward)
❌ Stop loss > 5% (too wide)
```
